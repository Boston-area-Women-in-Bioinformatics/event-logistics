Six phases

From idea to follow-up

The checklist walks you through every step. Here's what's covered.

01

Event concept

Shape your idea into a clear, compelling event people will want to attend.

Write a one-sentence hook
Draft the event description
Set capacity & target audience
Write the call to action
02

Logistics

Lock in the practical details — venue, date, speakers, and funding.

Confirm speakers or facilitators
Book venue or create virtual link
Lock the date and time
Identify a sponsor if needed
03

Promotion

Get the word out at least two weeks before the event.

Submit the BWiB promotion request form
Write the newsletter blurb
Draft the social media post
Prepare the full event description
04

Registration (Lu.ma)

Set up the event page in the BWiB Lu.ma workspace and publish it.

Create the Lu.ma event page
Add registration questions
Upload a cover image
Preview on mobile, then publish
05

Event day

Make sure everything runs smoothly on the day itself.

Send reminder email 24–48 hours before
Share prep materials with attendees
Test AV, virtual link, or room setup
06

Follow-up

Close the loop with attendees, sponsors, and your team.

Send a follow-up email within 48 hours
Include a short feedback form
Thank sponsors, speakers & volunteers
Share feedback with the organizing team
